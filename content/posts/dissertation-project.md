## 1. Project Overview
This project explores how caring touch can be computationally characterized using force sensing, machine learning, and multimodal data analysis. The methodology involves recruiting new parents as participants to interact with instrumented infant dolls equipped with force sensors, enabling the controlled collection of high-resolution tactile interaction data. Additionally, qualitative descriptive data will be gathered to help recognize and contextualize the data related to caring interactions. The research will involve designing data processing pipelines, exploring feature extraction and temporal modeling techniques, and evaluating machine learning approaches to uncover meaningful patterns in the data. The findings will provide essential knowledge for the development of intelligent haptic technologies and socially interactive robotic systems. The project will also develop innovative data analysis approaches with potential for publication.

## 2. Dataset
The dataset will be collected by tactile interaction data from instrumented infant dolls by 20 participants, followed by preprocessing and feature extraction to identify meaningful characteristics of caring touch.

## 3. Methodology
The project will involve collecting tactile interaction data from instrumented infant dolls, followed by preprocessing and feature extraction to identify meaningful characteristics of caring touch. Machine learning approaches, including supervised classification, unsupervised clustering, and temporal modelling techniques, will be explored depending on the structure of the collected data. Python-based tools such as Scikit-learn and PyTorch will be used for analysis and model development. The developed approaches will be evaluated using appropriate metrics such as classification accuracy, F1-score, clustering quality measures, and comparison with human annotations where available.
